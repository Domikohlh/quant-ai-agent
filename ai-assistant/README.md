# Personal AI Assistant

Private AI assistant for iOS, iPadOS, and macOS with RAG capabilities.

## Features
- Daily task planning
- Stock market intelligence via Alpaca
- Document Q&A with RAG
- Online/offline hybrid operation
- Controlled system access
